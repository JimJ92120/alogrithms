# diamond
| Languages |
|-----------|
| `PHP`     |

## Description
For a given input `n`, print a diamond like form with each vertices being of length `n`.  

e.g:
For `n` is `5`, the following is output (spaces being replaced by `=` for readability):
```
====*
===***
==*****
=*******
*********
=*******
==*****
===***
====*
```
