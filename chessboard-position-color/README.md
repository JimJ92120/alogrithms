# chessboard-position-color
| Languages |
|-----------|
| `PHP`     |

## Description
For a given position in a chessboard, find if the square is `light` or `dark`.   
  
![Chessboard](./chessboard.png) 
  
e.g:
- for a `a1`:
```php
// input
"a1"

// output
"dark"
```
- for `a2`:
```php
// input
"a2"

// output
"light"
```
